# Crafting reference (local)

Plain-text, in-repo copies of exact crafting mechanics so they're always referenceable.
Effects are datamined from poe2db.tw; bow weights from craftofexile. **Refresh:** `pnpm reference`.

| File | What |
|---|---|
| [currency.md](currency.md) | Orbs — incl. Greater (min mod lvl 35) / Perfect (min 50) exalt floors |
| [omens.md](omens.md) | Every omen's exact effect — Sinistral=prefix, Dextral=suffix |
| [bones.md](bones.md) | Desecration bones — min mod level + slot pool |
| [essences.md](essences.md) | Essence mechanic (Magic→Rare, one per item) |
| [bow-affix-weights.md](bow-affix-weights.md) | id_base 20 mod pool + per-slam odds by exalt floor |

Raw snapshot for the sim: `data/reference/bow-id20-mods.json`.
